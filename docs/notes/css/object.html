object-fit

  1. fill 默认值 不约束比例填满整个内容区域

  2. contain 保持长宽比缩放，保证元素完整显示，可能出现空白

  3. cover 保持长宽比缩放，保证内容区域被填满，同 background

  4. none 保持原尺寸和比例

  5. scale-down 等比缩
