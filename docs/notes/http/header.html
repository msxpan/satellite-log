Content-disposition

Content-disposition 是 MIME 协议的扩展，MIME 协议指示 MIME 用户代理如何显示附加的文件。当浏览器接收到头时，它会激活文件下载对话框，它的文件名框自动填充了头中指定的文件名

Content-Disposition: attachment; filename="name.jpg"
